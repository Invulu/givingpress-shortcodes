GivingPress Shortcodes v1.2
http://www.givingpress.com

A set of simple shortcodes for use with the GivingPress Theme. Examples and instruction for the shortcode usage can be found on the demo site, http://preview.givingpress.com/shortcodes/

INSTALL:
1. Upload the plugin by navigating to Plugins > Add New Themes within the WordPress admin. Select the plugin zip file to upload.
2. Activate the plugin after uploading.
3. Add the shortcodes within your page or post editor as seen in the demo site.
